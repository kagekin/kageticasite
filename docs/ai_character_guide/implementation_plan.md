# 実装計画：AIキャラクター作成ガイドの追加

Gemini と OiiOiiAI を組み合わせてオリジナルキャラクターを作成するガイドページを新規作成し、トップページおよび作品一覧からアクセスできるようにします。

## 変更内容

### 1. 新規ガイドページの作成

- **ファイル名**: `lab/oiioii_vtuber.html`
- **内容**: ユーザーから提供されたガイド内容を HTML 化します。
- **デザイン**: 既存の Lab ページに合わせた、SF的でプレミアムなデザイン（Tailwind CSS を使用）を適用します。
- **ポイント**:
  - Gemini でじっくり設定を作る方法に加え、いきなり OiiOiiAI にキャラの方向性を入れてお任せで作る「時短ルート」についても記載します。
- **構成**:
    1. Gemini での設定（魂の創造）
    2. OiiOiiAI での画像生成（姿の顕現）
    3. 動画化と読み上げ設定（命の吹き込み）
    4. 詳しいやり方へのリンク

### 2. トップページ (`index.html`) の更新

- 「作品・ツール」セクションに、このガイドへのリンクカードを追加します。
- 「Topics」セクションに、ガイド公開のお知らせを追加します。

### 3. 作品一覧ページ (`works.html`) の更新

- 「実験作・ツール」セクションに、ガイドへのリンクカードを追加します。

## 検証計画

### 自動テスト

- なし（HTML/CSS の静的更新のため）

### 手動確認

- ブラウザツールを使用して、各ページのデザインとリンクの遷移を確認します。
- モバイル表示での崩れがないか確認します。
- リンク先（OiiOiiAI 公式サイトなど）が正しく設定されているか確認します。
