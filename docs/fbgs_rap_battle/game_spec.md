## 1. コンセプト — 「文字で殴る。韻で刺す。」

### 1.1 ワンライン
**フォントの擬人化キャラが、自分のフォントで描かれたリリックを武器にラップバトルするゲーム。**

### 1.2 なぜ「フォント × ラップ」なのか

ラップバトルの本質は **「言葉の力」** — ライム（韻）、パンチライン（決めゼリフ）、フロウ（リズム）。
フォントの本質は **「文字の見た目」** — ウェイト（太さ）、セリフ（飾り）、カーニング（字間）、リガチャ（合字）。

この2つが重なるとき、**「言葉の意味」と「文字の形」の両方が武器になる** 唯一無二のゲームが生まれる。

- 太いブラックレターで刻まれた「DEATH」はズシンと重い一撃
- 軽やかなハンドライトで綴られた「kiss♡」はすばしっこいジャブ
- 完璧なライム（韻踏み）を決めれば、文字が画面上で爆発する

### 1.3 技術スタック
- HTML / CSS / Vanilla JavaScript（単一HTMLファイル）
- Web Audio API
- 配置: `game/fbgs_rap_battle/`

## 2. コア・ゲームメカニクス

### 2.1 ゲームの流れ（1バトル）

```
ビートが流れる（60〜90秒）
  ↓
[自分のターン] リリックを選んで投げる（攻撃）
  ↓
[相手のターン] 飛んでくるリリックに対処する（防御）
  ↓
（交互に繰り返し）
  ↓
HP先に0 or タイムアップ時HP多い方が勝ち
```

### 2.2 攻撃フェーズ — 「リリックを組む」

ビートに合わせて **「ワードパネル」** が画面下部に出現。プレイヤーは4つのパネルから1つを選んでリリックを発射する。

```
┌──────────────────────────────────────────┐
│                                          │
│    相手キャラ          自分キャラ          │
│      ←← リリック飛来    リリック発射 →→   │
│                                          │
│  ════════════ ビートライン ════════════   │
│                                          │
│  ┌──────┐ ┌──────┐ ┌──────┐ ┌──────┐   │
│  │ FIRE │ │ COLD │ │ WILD │ │ GOLD │   │
│  │  [D] │ │  [F] │ │  [J] │ │  [K] │   │
│  └──────┘ └──────┘ └──────┘ └──────┘   │
└──────────────────────────────────────────┘
```

#### ワードパネルの仕組み
- 毎ビートごとに4つの英単語がランダムに提示される
- プレイヤーはビートに合わせて1つ選んで押す（D/F/J/K）
- **選んだ単語が自分のキャラのフォントで描画され**、相手に向かって飛んでいく

#### フォントが攻撃力を決める

| フォント特性 | ゲーム上の効果 | 例 |
|------------|--------------|-----|
| **ウェイト（太さ）** | **攻撃力**。太いフォントほど1発が重い | ブラックレター系 → 高ATK / ハンドライト系 → 低ATK |
| **字幅（文字の幅）** | **ヒット範囲**。幅広フォントは避けにくい | 幅広セリフ → ガード困難 |
| **字間（カーニング）** | **コンボ速度**。詰まったフォントほど連射が速い | サイバーサンセリフ → 高速連射 |
| **装飾（セリフ / スワッシュ）** | **特殊効果の発動率**。装飾が多いほどクリティカル率UP | ファンタジックセリフ → クリティカル率高 |

#### ライム（韻踏み）ボーナス 🔥

**連続で韻を踏んだ単語を選ぶとボーナスダメージ！**

例:
- 1ビート目: "FIRE" → 通常ダメージ
- 2ビート目: "WIRE" → ライム成立！ **ダメージ×1.5 + 画面にライムエフェクト**
- 3ビート目: "HIGHER" → 3連ライム！**ダメージ×2.0 + 文字が巨大化して爆発**

```
ライムチェイン: FIRE → WIRE → HIGHER
                ×1.0   ×1.5    ×2.0
                          🔥 RHYME CHAIN!! 🔥
```

**ゲームがライム候補を仕込む**: 4つのパネルのうち1〜2個は前のワードと韻を踏める単語が含まれる。プレイヤーが見抜いて選べばボーナス。

### 2.3 防御フェーズ — 「相手のリリックを捌く」

相手のターンでは、相手のフォントで描かれた単語が飛んでくる。

| アクション | 操作 | 効果 |
|-----------|------|------|
| **ガード** | タイミングよくSpaceを押す | ダメージ半減。相手フォントのリリックが砕けるエフェクト |
| **カウンター** | 飛来するリリックと同じ韻の単語をD/F/J/Kで選ぶ | **ダメージ無効化 + 反撃ダメージ！** 韻で返すのがラップバトルの醍醐味 |
| **ミス** | 何もしない | フルダメージ |

### 2.4 パンチライン（必殺技）

攻撃ヒット or ライムボーナスで **「HYPE ゲージ」** が溜まる。MAXになると **パンチライン** が使える。

- **発動**: Enterキー
- **演出**: キャラ固有の英語パンチライン（1文）がキャラのフォントでフルスクリーン表示
- **効果**: 大ダメージ + 画面全体がキャラのフォントのグリフで埋め尽くされる

例（NECLA発動時）:
```
画面全体がNeoClassicalGothフォントで覆われる

     ╔════════════════════════════════╗
     ║                                ║
     ║   "Your soul is my SERIF."     ║
     ║                                ║
     ║   — NECLA / NeoClassicalGoth   ║
     ╚════════════════════════════════╝

→ ブラックレターの文字が画面中に拡散して相手を包囲
```

### 2.5 フォント属性ごとの戦闘スタイル

| フォント属性 | ラップスタイル | ATK | SPD | DEF | 特殊能力 |
|------------|-------------|-----|-----|-----|---------|
| **ブラックレター** | ハードコア / ダークラップ | ★★★ | ★ | ★★ | 「ヘビーグリフ」: 太い文字が画面に残留し追加ダメージ |
| **サイバーサンセリフ** | テックラップ / トラップ | ★★ | ★★★ | ★ | 「オートカーニング」: コンボが途切れにくい |
| **ハンドライト** | オルタナ / ポップラップ | ★ | ★★ | ★★ | 「スクリブル」: ランダムで相手のパネル文字を読みにくくする |
| **ファンタジックセリフ** | ソウル / ジャジーラップ | ★★ | ★★ | ★★ | 「オーナメント」: ライム成功時の倍率が他より高い |
| **ベクターハーツ** | エレクトロ / フューチャーベース | ★★ | ★★ | ★★ | 「グリッチフォント」: 相手の画面にノイズを走らせる |
| **ニュートラルスクリプト** | フリースタイル | ★★ | ★★ | ★★ | 「アダプト」: 相手のフォント属性を一時コピーできる |

- 「FBGS RAP BATTLE」ロゴ（グラフィティタイポ）

- キャラカードにフォントサンプル（実際のフォントのアルファベット数文字）を表示

## 3A. シナリオパート — 「フォントたちの言葉」

### 3A.1 シナリオモードの概要

ストーリーモードでは、バトルの前後に **シナリオパート（会話シーン）** が入る。
フォントの擬人化キャラたちは「文字の存在」であるため、**独特の喋り方** をする。

### 3A.2 フォントキャラの喋り方

#### 表示スタイル: 「カタカナ＋英語」二段構成

キャラのセリフは **上段にカタカナ / 下段に英語** の2行で表示される。カタカナ表記は「文字（フォント）の存在が人間の言葉を真似ている」ようなぎこちなさ・独特のリズム感を持つ。英語はキャラのフォントで描画される。

```
┌─────────────────────────────────────────────┐
│                                             │
│   [NECLA のアイコン]                         │
│                                             │
│   ワレワレハ モジ ダ。                        │
│   ニンゲン ノ コトバ ヲ カタドル モノ ダ。      │
│                                             │
│   "We are LETTERS.                          │  ← NeoClassicalGothフォントで表示
│    We are the ones who SHAPE human words."  │
│                                             │
│                          [ ▶ NEXT ]         │
└─────────────────────────────────────────────┘
```

#### カタカナ口調のルール

| ルール | 説明 | 例 |
|-------|------|-----|
| **全文カタカナ** | 漢字・ひらがなは使わない。助詞もカタカナ | 「ワタシ ハ モジ ダ」 |
| **単語区切りスペース** | 単語ごとにスペースを入れて区切る（テキストデータっぽさ） | 「コノ バトル デ キメル」 |
| **フォント用語が混ざる** | 日常会話にフォント用語が自然に出てくる | 「オマエ ノ カーニング、アマイゼ」（＝お前の間合い、甘いぜ） |
| **感情表現が記号的** | 感情をテキスト記号や書体用語で表す | 「…（ボールド）！！」（＝怒り） / 「イタリック…」（＝照れ） |
| **キャラごとの癖** | 属性に応じた語尾・口調の差 | ブラックレター系: 重厚な言い回し / ハンドライト系: 軽くポップ |

#### キャラ別の口調例

| キャラ | カタカナ口調の特徴 | 例 |
|-------|-----------------|-----|
| **NECLA** | 低温。句点が多い。ゴシック的な荘厳さ | 「…シズカ ニ シロ。ワタシ ノ グリフ ガ オマエ ヲ キザム。」 |
| **2TON** | テンション高め。ストリート調 | 「ヨッシャ！ ツギ ノ デリバリー ハ コノ リリック ダゼ！」 |
| **PACE** | ギャル語混じり。語尾に♡がつく | 「マジ？ ソノ フォント ウェイト、カルスギナイ？♡」 |
| **BLANC** | 静かで詩的。間合いが長い | 「タマシイ ノ セリフ…… ソレ ガ ワタシ ノ カタチ。」 |
| **GRIRE** | 荒々しい。脅し調 | 「オラオラ！ ワタシ ノ シニガミ フォント デ ケシテ ヤル！」 |
| **AXEL** | 理系っぽい。分析的 | 「フム。 オマエ ノ ベースライン、 0.3px ズレテル。」 |

### 3A.3 ストーリーモード構成（MVP案）

プレイヤーは1人のキャラを選んで、フォントの世界を舞台にしたストーリーを進める。

#### あらすじ（仮）

> フォントたちが暮らす「タイプフェイス・シティ」。
> ある日、謎のバグフォント「████」が出現し、街のフォントたちの文字データを壊し始めた。
> 文字が壊れる ＝ 自分の存在が消える。
> フォントたちは自分のリリック（言葉の力）で戦い、街を守ることを決意する。
>
> 「モジ ガ コワレル ナラ、 モジ デ タタカウ マデ ダ。」

#### シナリオの流れ（MVP: 3ステージ）

| ステージ | シナリオ | 対戦相手 |
|---------|---------|---------|
| **Stage 1** | プレイヤーキャラが異変に気づく。仲間と出会い、ラップバトルで実力を確認 | 仲間キャラ（フレンドリーマッチ） |
| **Stage 2** | バグの影響が広がる。別属性のキャラと衝突、意見の対立からバトルへ | 別属性のライバルキャラ |
| **Stage 3** | バグフォントの正体に迫る。最終決戦 | ボスキャラ「████」 |

## 6. ビジュアル & サウンド

### 6.1 ビジュアル方針
- ダークベース + キャラカラーのネオン光
- **飛び交うリリックは各キャラのフォントで描画** — これが最重要
- ヒット時にフォントのグリフが砕け散る / 炎上する / 跳ね返るパーティクル
- 背景: コンクリート壁 + グラフィティ（コンボで壁にタグが増える）
